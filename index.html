ChronoLedger – Secure Freelancer Task Logging System

ChronoLedger is a Java-based desktop application designed to securely track, verify, and audit freelance or remote developer activity. It provides a tamper-proof digital ledger using cryptographic hash chaining to record each work session. Users can log tasks with timestamps, and the system ensures ledger integrity through hash verification. Features include PDF/CSV export, integrity check alerts, and per-user persistent logs. A JavaFX-powered GUI offers a smooth and modern user experience. The application is particularly suited for remote teams, freelancers, or employers seeking transparent work validation. It helps ensure that billed hours are matched by actual logged work—mitigating fraud and increasing trust in remote collaboration.

Java OOPs Data Structure SHA-256 Hashing UI-UX File Handling
